# 基于网易我的世界API的村庄范围检测方案

## 检测原理
通过生成实体，利用环境过滤器的`in_village`条件来动态增删组件组，再通过监听组件组的变换来判断村庄范围。

## 存在的难点
当前方案仅能检测单个村庄，对于多个村庄融合的情况，检测会出现异常，处理难度较大。
